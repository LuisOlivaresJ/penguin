Welcome!

Penguin is a web application used to track the position reproducibility of an image plane detector (or electronic portal imaging device, EPID) used to acquire x-ray images with a linear accelerator (LINAC).

It is intended for medical physicists who want to follow AAPM TG 307 recommendations about tests to ensure the EPID will function accurately as a dosimeter.

Note: This app is the result of my final project from the Harvard University CS50x course.
